┌──────────────────────────────────────────────────────────────────────────┐
│ 🔬 WHY TUSTIN > ZOH FOR 1M+ TOKENS                                       │
└──────────────────────────────────────────────────────────────────────────┘

  Mathematical Advantage:

  1. Conformal Mapping
     ZOH:    s-plane → z-plane (non-conformal)
     Tustin: Entire left s-plane → unit circle (conformal)

     ⟹ Stability GUARANTEED for all stable continuous systems

  2. Numerical Stability
     ZOH:    Ā = exp(Δ·A)     → exp(-1000) = underflow at 1M steps
     Tustin: Ā = (1+x)/(1-x)  → always bounded, no underflow

  3. Frequency Response
     ZOH:    Magnitude warping at high frequencies
     Tustin: Magnitude preserved, only phase warping

  4. Gradient Flow
     ZOH:    Gradients vanish through exp() for long sequences
     Tustin: Gradients stable through rational function

  Practical Impact for hg38:

  | Metric              | ZOH          | Tustin       |
  |---------------------|--------------|--------------|
  | Max stable length   | ~100K tokens | 1M+ tokens   |
  | bfloat16 training   | Unstable     | Stable       |
  | Long-range deps     | Poor         | Excellent    |
  | Gradient flow       | Degrades     | Consistent   |

│ 🚀 HOW TO USE RC EQUIVARIANCE                                            │

  Option 1: Drop-in Data Loader (Easiest)

    from rc_equivariance import RCDataLoader

    # Wrap existing loader
    base_loader = HG38DataLoader(config)
    rc_loader = RCDataLoader(base_loader, mode='random')

    # Use as normal (automatically applies RC)
    x, y, mask = rc_loader.get_batch('train')

  Option 2: Manual Augmentation

    from rc_equivariance import rc_augment_batch

    x, y, mask = loader.get_batch('train')
    x, y, mask = rc_augment_batch(x, y, mask, rng_key)

  Option 3: RC Consistency Loss

    from rc_equivariance import rc_consistency_loss

    # Add to training step
    ce_loss = standard_loss(...)
    rc_loss = rc_consistency_loss(model_fn, params, x)
    total_loss = ce_loss + 0.1 * rc_loss
